## 1. 简介

iToDo 是一款基于 Microsoft ToDo 设计的任务管理应用，支持以下功能：

- 创建和管理任务
- 标记任务重要性和完成状态
- 按清单分类任务
- 多种筛选视图（我的一天、已计划、重要、已完成）
- 任务导入导出
- 多语言支持（中文/英文）
- 深色/浅色主题

## 2. 快速开始

### 2.1 界面概览

> [截图：主界面标注各个区域]

## 3. 任务管理

### 3.1 标记任务为重要

点击任务标题旁边的 **星标图标** 可将任务标记为重要。

> [截图：标记重要的星标图标]

- 黄色星标 = 重要任务
- 灰色星标 = 普通任务

### 3.2 完成任务

点击任务左侧的 **复选框** 可标记任务完成。

> [截图：完成前后对比]

- 未完成：空心复选框
- 已完成：实心勾选 + 删除线

### 3.3 编辑任务

1. 点击任务标题打开右侧详情面板
   > [截图：右侧详情面板]

2. 在详情面板中可以编辑：
   - 任务标题
   - 任务内容/备注
   - 截止日期
   - 开始日期
   - 提醒时间
   - 重复规则

### 3.4 删除任务

**方法一：右键删除**
1. 在任务上点击鼠标右键
2. 在弹出的菜单中点击 **"删除"**
   > [截图：右键菜单]

**方法二：详情面板删除**
1. 点击任务打开详情面板
2. 点击底部的 **"删除任务"** 按钮
   > [截图：删除按钮位置]

### 3.5 添加子任务

1. 点击任务打开详情面板
2. 在 **"子任务"** 部分点击 **"添加子任务"**
3. 输入子任务内容后按 Enter 完成
   > [截图：子任务列表]

## 7. 导入导出

### 7.1 导出任务

1. 点击设置 → 鼠标悬停在 **"导出"** 上
2. 选择要导出的清单（或"全部任务"）
   > [截图：导出菜单]

3. 在弹出的文件对话框中选择保存位置
   > [截图：文件保存对话框]

4. 点击保存，任务将导出为 JSON 文件

### 7.2 导入任务

1. 点击设置 → **"导入"**
   > [截图：导入菜单]

2. 在弹出的文件对话框中选择要导入的 JSON 文件
   > [截图：文件选择对话框]

3. 点击打开，任务将被导入

> **注意**：
> - 导入时会自动创建不存在的清单
> - 导入的任务会使用新的 ID

## 8. 常见问题

### Q1: 如何备份我的任务？

使用导出功能将任务导出为 JSON 文件进行备份。

### Q2: 任务数据存储在哪里？

任务数据存储在以下位置：
- macOS: `~/Library/Application Support/com.itodo.iToDo/itodo.db`

### Q3: 如何更新关于信息？

编辑应用目录下的 `src-tauri/about.json` 文件，然后重新打包应用。

### Q4: 提醒功能如何使用？

在任务详情面板中设置提醒时间，系统会在指定时间发送通知。
