## Adding Screenshots

To add screenshots to the documentation:

1. Take screenshots of the app interface
2. Save them in the `docs/images/` directory
3. Update the documentation files to reference the screenshots

### Screenshot Naming Convention

Use descriptive names:
- `main-interface.png` - Main interface overview
- `add-task-button.png` - Add task button location
- `task-created.png` - Task created example
- `right-click-menu.png` - Right-click menu
- etc.

### Recommended Screenshot Sizes

- Full interface: 1200x800 pixels
- Partial UI elements: 400x300 pixels
- Icons and buttons: Actual size or 2x for retina
